# Bachelor Thesis

A web-based video conferencing application developed as a bachelor thesis, built with React and modern web technologies. The application supports real-time video calls, messaging, screen sharing, and user authentication, with emphasis on usability and extensibility.

---

## Features

- User authentication (Email/Password, Google, Facebook)
- Real-time video conferencing
- Live chat during meetings
- Screen sharing
- Call creation and joining via unique Call ID
- Responsive UI for desktop and mobile

---

## Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS
- Firebase (Authentication & Firestore)
- GetStream.io Video API

---

## Getting Started

### Prerequisites
- Node.js (LTS)
- npm
- Git

### Installation
```bash
git clone https://github.com/mmtzitziras/auth-thesis.git
cd auth-thesis
npm install
npm run dev
```

## Notes

This project is provided as a **demo application** for academic and portfolio purposes.  
Sensitive configuration data such as **API keys and Firebase credentials are not included in the repository**.

To run the project locally, you must create your own configuration files and provide your own keys.
